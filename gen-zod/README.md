## Installation

Install Deno

```bash
# On MacOS/Linus
curl -fsSL https://deno.land/install.sh | sh

# On Windows
irm https://deno.land/install.ps1 | iex
```

Install Skmtc

```bash
deno install -g -A jsr:@skmtc/cli -n skmtc -f
```

## Create project and generate artifacts

```bash
# Run Skmtc and follow prompts
skmtc
```
